Projects

Method

Applied to every project, so any of them can be handed to another engineer without a conversation.

Measured, not asserted

Every performance claim has a validation report with the method, the sample size, and a statement of what the numbers don't prove.

Documented for handoff

Technical documentation in ASD-STE100 Simplified Technical English, organised by the Diátaxis model.

Decisions recorded

Significant technical choices are written as ADRs: context, decision, and the negative consequences as well as the positive.

Tested at the formula level

Golden tests pin the mathematics against the authoritative source, so a model cannot silently drift.

Risks stated openly

A risk register with likelihood, impact, and mitigation — including the risks that are simply accepted.

Built to be updated

Anything that changes over time lives in one configuration block, so a new version is a single edit.
