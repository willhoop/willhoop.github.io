# Security and Privacy

**Version 1.0 · Last updated 2026-07-22**

## Scope
the portfolio site runs entirely in the browser or as a static site. It has no backend, no accounts,
and no server that stores user data.

## What it reads and where data goes
- Only public data (game data, public market or replay data) and, where applicable, values
  you enter yourself.
- All computation is local to your browser. Nothing is transmitted to a server owned by
  this project.

## Reporting a vulnerability
Email **willjhooper@msn.com** with a description and steps to reproduce. Please do not open
a public issue for a security problem before it is fixed. Expect an acknowledgement within
a few days.

## Dependencies
Third-party libraries are pinned and loaded from reputable CDNs. Update them when an
advisory affects a pinned version.
