Pokémon — Competitive VGC Program

An umbrella project for competitive Pokémon VGC work: modelling, tooling, and analysis for the Champions VGC 2026 Reg M-B format on Pokémon Showdown.

Sub-projects

Sub-project What it does Status
CHOMP Champions Head-to-head Optimizer for Matchup Prediction. Recommends which four Pokémon to bring and which two to lead at team preview, using real damage mathematics. Working, in daily use
(planned) Turn simulator Multi-turn lookahead. Adds recoil, flinch, status, and redirection. Roadmap
(planned) Usage model Opponent set inference from measured usage distributions. Roadmap

Each sub-project is self-contained and carries its own full document set (white paper, plain-English deck, ASD-STE100 technical documentation, validation report, risk register, ADRs).
